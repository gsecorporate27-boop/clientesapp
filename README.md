# CodexApp RIV

Aplicacion Vite + React para la Ruta de Implementacion Visible.

## Ejecutar localmente

```bash
npm install
npm run dev
```

## Publicar en Vercel

- Framework preset: Vite
- Build command: `npm run build`
- Output directory: `dist`

La app puede recibir el Google Sheet por URL:

```text
?sheet=ID_DEL_GOOGLE_SHEET
```
