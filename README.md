# Ruta de Avance Visible™ - Premium Homogénea V2

## Cambios principales

- Se mantiene la primera pestaña `Portal del proyecto`.
- Todas las pestañas ahora tienen una línea visual más premium y homogénea.
- La paleta se basa únicamente en el color principal de GSE:

```text
#00b8b5
```

## Mejoras visuales

- Tarjetas destacadas con degradado turquesa.
- Texto blanco en tarjetas premium.
- Menos tonos externos y más consistencia visual.
- Mejor margen, profundidad visual y jerarquía.
- Resumen, Ruta, Hallazgos, Pendientes, Entregables y Lo que vas a recibir quedan visualmente alineados al portal.
